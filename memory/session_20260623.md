## 작업 개요

NAMU Agent 시스템의 1단계(개인용) 핵심 구조를 완성했다.

## 만든 것

### 어댑터 레이어 (`adapters/`)

| 파일 | 설명 |
|------|------|
| `base.py` | `AIAdapter` ABC — 모든 어댑터가 구현해야 할 인터페이스 |
| `claude_api.py` | Anthropic Claude API 어댑터 (`anthropic` SDK) |
| `gpt_api.py` | OpenAI GPT API 어댑터 (`openai` SDK) |
| `ollama.py` | Ollama 로컬 모델 어댑터 (OpenAI 호환 `/v1` 엔드포인트) |
| `claude_subscription.py` | Claude 구독 계정 어댑터 (`claude-agent-sdk`) |

**우선순위:** `ollama`(1) → `claude-subscription`(2) → `claude-api`/`gpt-api`(3)

### 오케스트레이터 (`core/orchestrator.py`)

흐름: 어댑터 선택 → **승인 게이트** → 실행 → 파일 기록

- `run()` — 작업 실행 진입점
- `select_adapter()` — priority 순으로 `is_available()` 통과한 첫 번째 어댑터 선택
- `_approve()` — 워커 호출 전 사용자 y/N 확인
- `_write_task_files()` — `tasks/{id}/task.md`, `result.md`, `log.md` 생성
- `_append_learning()` — `memory/learnings.md` 성공/실패 섹션에 자동 추가

### 설정 (`config.py`)

현재 활성화된 어댑터: `claude-subscription`, `claude-api`

## 주요 설계 결정

### ClaudeAPIAdapter
- adaptive thinking 기본 적용 (`thinking: {type: "adaptive"}`)
- thinking 사용 시 temperature를 1.0으로 강제 (API 요구사항)
- 스트리밍 + `.get_final_message()`로 타임아웃 방지

### ClaudeSubscriptionAdapter
- `claude-agent-sdk`의 `query()` 함수 사용 (async → `asyncio.run()` 래핑)
- `tools=[]`, `permission_mode="dontAsk"`, `max_turns=1` — 순수 텍스트 응답
- `chat()`: `Role.SYSTEM` 메시지를 `system_prompt`로 분리, 나머지를 직렬화해 단일 프롬프트로 전달
- `is_available()`: `shutil.which("claude")` + `~/.claude` 디렉토리 존재 여부

### GPTAPIAdapter
- `openai.OpenAI` 클라이언트를 지연 초기화 (`_get_client()`)
- API 키 없이도 인스턴스 생성 가능 → `is_available()` false 반환으로 자연스럽게 스킵

### OllamaAdapter
- `openai.OpenAI(base_url=f"{host}/v1", api_key="ollama")` — 새 의존성 없이 구현
- `is_available()`: `models.list()`로 지정 모델이 실제로 pull 되어 있는지 확인

## 실행 테스트 결과

### claude-subscription 단독 테스트
```
generate('\"안녕\" 이라고만 대답해줘')
→ content='안녕', model=claude-sonnet-4-6, 958/1 tokens

chat([SYSTEM: 숫자만 대답, USER: 2+3은?])
→ content='5', model=claude-sonnet-4-6
```

### Orchestrator 테스트
```
run('피보나치 수열 10번째까지 출력하는 코드 짜줘', require_approval=False)
→ 성공, elapsed=3.47s
→ tasks/20260623_221353_22fdae/ 생성 (task.md, result.md, log.md)
→ memory/learnings.md 성공 패턴 섹션 자동 기록
```

### 승인 게이트 테스트
```
입력 "y" → 실행 진행
입력 "n" → RuntimeError('사용자가 작업을 취소했습니다.'), API 호출 없음
```
