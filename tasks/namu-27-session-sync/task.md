## 목적
사용자가 CC를 켜기 전 git pull을 잊어도 NAMU가 스스로 최신·정확한 상태로 세션을 시작하게 한다. 07-10 실증된 3중 실패를 구조적으로 해소: ① pull 전 낡은 브리핑 ② 마감 task가 context 미닫음으로 유령 활성 판정(#25 재발) ③ 모든 task 마감 시 이월 항목 안내 부재.
